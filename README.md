🎵 Anti-Gravity Musical Instrument Detector

 🚀 Project Overview
This project is an AI-based system that detects musical instruments from audio input and displays the result using a floating (anti-gravity) UI.

 🎯 Features
- 🎧 Audio input processing (microphone)
- 🤖 Machine learning model for classification
- ⚡ Real-time instrument detection
- 🌌 Anti-gravity floating UI output


🛠️ Technologies Used
- Python
- Librosa (audio processing)
- TensorFlow (ML model)
- Scikit-learn
- Matplotlib (UI visualization)

 📂 Project Structure
anti-gravity-music-detector/
│── features.py
│── train_model.py
│── predict_live.py
│── ui.py
│── requirements.txt
│── dataset/ (not uploaded)

▶️ How to Run

 1. Install dependencies
pip install -r requirements.txt

 2. Train model
python train_model.py

 3. Run detection
python predict_live.py

 📸 Output
- Detects instrument (Guitar / Piano)
- Displays result in floating UI

 🎯 Applications
- Music learning tools
- Smart classrooms
- Interactive demos
